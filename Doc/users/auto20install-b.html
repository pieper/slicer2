Compatibility

Slicer has been tested on a variety of Windows versions and graphics cards, but you may encounter problems depending on your system configuration. Slicer was compiled and most completely tested under Windows 2000. WinZip must be installed on your computer before you begin the installation process.

System Requirements

  • Memory: Minimum: 128 MB; recommended: 256 MB or greater.
  • Disk Space: 11.5 MB to download the installer, which expands to about 45 MB. The tutorial data set requires 37 MB.
  • Processor Speed: Has run successfully on a 333-MHz machine with Windows 98; a faster processor speed is recommended.
  • Graphics Card: Some older video cards and some laptops restrict the amount of memory allocated to graphics functions and may result in reduced image quality.
  • Screen Resolution and Colors: Minimum screen resolution: 1024 x 786; recommended: 1280 x 1024.
    Minimum colors: 16 bit color; prefer: 32 bit color. (Note: 24 bit color works on most machines, but may cause some rendering problems.)

Installation Steps

  1. If you already have an older version of Slicer installed, uninstall it. (You can uninstall Slicer using the standard tools in the Windows Control Panel. For example, with Windows 2000, from the Start menu, select Settings, then Control Panel, then Add/Remove Programs, and then select the version of Slicer to uninstall. If this does not work, simply delete the Slicer program files. There are no Registry entries to change.)

    2.  
  2. Download the Windows version of Slicer onto your computer. (Instructions about how to download the most recent version were emailed to you upon receipt of your signed Software License Agreement. If you have not filled this out yet, click here.)

    3.  
  3. Double-click the self-extracting executable file to start the installation procedure.

    When the installation has completed, a Slicer shortcut will appear in the Start menu, under Programs.

    Note: The default directory for storing MRML files is the installation directory. If you wish to specify a different location in which to store your MRML files, right-click the Slicer shortcut and select Properties. Set the Start in: field to the directory where you store your MRML files and click OK.

    4.  
  4. Set the environment variable SLICER_HOME to the directory where you installed Slicer. For example, C:\Program Files\Slicer.

    5.  
  5. You are now ready to run Slicer by selecting Start, then Programs, then Slicer.

Frequently Asked Questions
What if I need more detailed instructions for installing with Windows NT?
Look in the Developer's Guide, under WinNT Install
How can I always load a particular data set?
Copy the original Slicer shortcut, then paste it to create another shortcut. Right-click the new copy, and select Properties. In the Target text box, type the name of the MRML file you wish to open. Then click OK. You can change the name of the new shortcut by selecting the name, clicking the name once, typing the new name, and clicking Enter.
How can I uninstall Slicer?
Use the standard tools in the Windows Control Panel to remove the version of Slicer that you no longer wish to use. For example, with Windows 2000, from the Start menu, select Settings, then Control Panel, then Add/Remove Programs, and then select the version of Slicer to uninstall.
