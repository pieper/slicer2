There are several ways to save data in Slicer.

  • Use Save Scene, Save Scene As, or Save Scene with Options (File submenu) to save a scene as an XML file. The XML file lists the elements of the scene and their attributes.
  • Use Save 3D View or Save 3D View As (File submenu) to save a picture of the 3D view as a TIFF file.
  • Use Save Active Slice or Save Active Slice As (File submenu) to save one of the slices as a TIFF file.
  • Use Save Current Options (File submenu) to save user preferences, like background color or view mode, into a special Options.xml file.
  • Use Save (Editor button ==> Volumes panel ==> Save) to save label maps.
  • Use Save (ModelMaker button ==> Save panel ==> Save) to save models.
