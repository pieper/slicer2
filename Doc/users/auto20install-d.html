Installing on Mac OS X

Mac OS X Installation Instructions

Compatibility

Slicer is compatible with Mac OS X, but it is not supported.
A test version is available here.

System Requirements

Slicer requires Mac OS 10.2 or greater.

Installation Steps

There is no installation. Open the archive and run the executable from inside the X server.
