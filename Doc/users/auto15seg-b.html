To segment data:
  1. On the Main menu, click Editor. The Editor panel appears.
  2. Click the Volumes tab. The Volumes tab appears.
  3. Click Setup. The name you assigned to the original series of data should be present in the Original Grayscale field. These are the images you will segment.
  4. Select NEW for the Working Labelmap, and enter a name for this segmentation. For example, if you are segmenting the ventricles of the brain, you might name this segmentation "ventricles".
  5. Click Start Editing. The Effects tab should appear.
  6. There are several tools that you may use to segment your original images, including:
    • Draw: Allows you to overlay the image with polygons.
      Draw is often useful with a well-defined tumor, for example.
      1. On the Draw panel, select an output color. This color will outline the tumor.
      2. For the Mode, select Draw.
      3. To apply this effect to only one slice/image of the original data, select 1 slice for Render.
      4. For the Shape, select Polygon.
      5. Use the mouse to encircle the section of the image to segment.
      6. Click Apply. The polygon you created should connect itself and overlay the tumor in the segmented slice.
      7. Continue this process for each of the original grayscale images.

      Note: If you encircle too much of an image in a particular slice, use Draw with Black output to create polygons around the extraneous areas; this acts like an eraser. You may also delete the last selected point of your polygon by selecting Delete Selected.

    • Threshold: Automatically selects and outlines various structures based on a certain voxel value.
      Threshold is very useful for brain, ventricles, and skin models of the head for example.
      1. On the Threshold panel, adjust the sliding scales to select only the structures you desire. The outlined areas are called "islands".
      2. Check some of the original data images to ensure that you are segmenting only what you want.
      3. Click Apply. The function will segment the selected areas.

    • Change Island: Changes an outlined area, an "island", to the color that you select.
      Change Island is very useful after thresholding. For example, you can eliminate unwanted areas by using a Black output and clicking on the areas to eliminate.


    • Remove Islands: Automatically removes areas based on size.
      Remove Islands is useful for removing small areas that should not have been thresholded.
      For example, if you use a value of "10", all islands less than 10 voxels in size will be removed.

  7. Save your segmented data (called a "label map").
    1. On the Main menu, click Editor. The Editor panel appears.
    2. Click the Volumes tab. The Volumes tab appears.
    3. Select Save.
    4. Select the name of your label map for Volume to Save.
    5. Select a file name. It is often simplest to use the same name as the Volume to Save.
    6. Save this file to the appropriate directory.
