Compatibility

Slicer is compatible with Solaris 2.8 (also known as Solaris 8 or SunOS 5.8) and OpenGL 1.2. Although we do not formally support Solaris 2.5, some users have been able to make it work with some difficulty.

System Requirements

  • Memory: Minimum: 128 MB.
  • Disk Space: 35.6 MB to download the installer, which expands to about 145 MB. The tutorial data set requires 37 MB.
  • Processor Speed: Has run successfully on a 450-MHz machine; a faster processor speed is recommended.

Frequently Asked Questions





    
How can I load an existing data set?

    
At the shell prompt, type slicer2 followed by the name of your MRML file.
