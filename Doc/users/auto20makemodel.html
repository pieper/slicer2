The output of the segmentation process is a set of label maps, where pixels take on values corresponding to tissue type. To create three-dimensional models, the bounding surfaces of the label maps are extracted and represented as a collection of triangles using marching cubes (a smoothing algorithm). Decimation then reduces the number of triangles to a quantity that can be more quickly rendered with little observable loss in detail. For example, a typical brain surface is reduced from approximately 500,000 triangles to 150,000.

To create a model from a label map:

  1. Click ModelMaker on the Main menu.
  2. On the Create tab of the ModelMaker panel, use the Volume pull-down menu to select the label map to use.
  3. The largest label in this volume is automatically selected. To make a model from another pixel value, either click Label to select a color, or enter the label value in the field.
  4. Enter the Name for the model.
  5. Change the values of the Smooth and Decimate parameters, if necessary.
  6. Click Create and wait for the model to be created. The model will appear in the 3D Viewer.
  7. To save the model:
    1. Click the Save tab in the ModelMaker panel.
    2. Select the model from the pull-down menu.
    3. Optionally enter a file name.
    4. Click Save. A dialog box will appear. Save it to the appropriate directory. The file will be saved as a .vtk file.
  8. To save the scene, including the new model:
    1. Click File.
    2. Click Save Scene or Save Scene As.
The model should now be created and will appear in the 3D Viewer. You can now:
  • Rotate the model with the left mouse button.
  • Translate the image with the middle mouse button.
  • Zoom the image with the right mouse button.
  • Hide and unhide the model, by clicking the name of your model on the Models panel.
  • Change the opacity of the model, with the sliding scale next to the name of your model on the Models panel.
