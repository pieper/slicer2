There are two ways to save the 3D view in the TIFF file format.

To save the 3D view:

  1. Click File. A drop-down menu appears.

  2. There are two ways to save the 3D view, depending on your preferences for file name.
    • To save the file with the default file name, select Save 3D View.
    • To use another file name, select Save 3D View As and enter your own file name.
