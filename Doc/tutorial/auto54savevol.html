Labelmap volumes are saved in the same place they are created, in the Editor.

  • First click the Editor button in the Main Menu. This will bring you to the Volumes tab (yes, the one inside the Editor).
  • Click the Save button (to the upper right inside the Editor->Volumes tab).
  • Select the labelmap you would like to save from the menu and click Save.
  • Then a Save Volume dialog box pops up. Choose the path you would like, and hit Save.
