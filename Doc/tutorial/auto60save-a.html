To save a description of the scene you are viewing, click on File in the pulldown menu. The first time, click on Save Scene As to choose a filename. Then to save later changes, click on Save Scene.

Save Scene With Options will include Options in the scene description. These are View Presets, background color, and other preferences you have set in this Slicer session.

Note: If you hit Save Scene the first time you save, without choosing a filename first through Save Scene As, a MRML file will be saved with the default name of Data.xml in the current directory.

To open the scene again later: Click on File, then Open. Find the file you saved, double click it, and wait for the volumes and models to load. Congratulations!

   

What is the Deal Here?

When you save a MRML file, you are saving the "scene." This means that you are saving the pathnames to the volumes and models, not the volumes and models themselves. This is great because it allows you to open them all again, just by opening the MRML file!
