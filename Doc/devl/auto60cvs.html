CVS Access

For anonymous CVS access, in order to check out the Slicer 2 code and documentation, you must set the environment variable CVSROOT:

setenv CVSROOT :pserver:anonymous@cvs.spl.harvard.edu:/projects/cvs/slicer
cvs login
cvs co slicer2

The password for anonymous login is bwhspl.

To request write access to the repository, please email slicer@ai.mit.edu.
Access to the web interface to CVS is available via the following URL: http://splweb.bwh.harvard.edu:8000/cgi-bin/develop/viewcvs.cgi/slicer2/?cvsroot=slicer
