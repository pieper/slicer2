For anonymous CVS access, in order to check out the Slicer 2 code and documentation, set the environment variable CVSROOT:

setenv CVSROOT :pserver:anonymous@cvs.spl.harvard.edu:/projects/cvs/slicer
cvs login
cvs co slicer2

The password for anonymous login is bwhspl.

To request write access to the repository, please email slicer@ai.mit.edu.
Please be aware, that according to the software license that applies to the Slicer code, that the copyright of code contributed to the cvs repository is signed over to the copyright holder of Slicer. Please see the Software License Agreement for further details, as it also applies to the source code, not just the distributed versions.
Access to the web interface to CVS is available via the following URL: http://splweb.bwh.harvard.edu:8000/cgi-bin/develop/viewcvs.cgi/slicer2/?cvsroot=slicer
