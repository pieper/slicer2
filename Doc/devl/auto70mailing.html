To subscribe to the developer mailing list, send mail to slicer-devel-request@bwh.harvard.edu with the word subscribe as the subject.

The address of the mailing list itself is slicer-devel@bwh.harvard.edu.

From inside the SPL, the web interface can be accessed through https://www.spl.harvard.edu/mailman/listinfo/slicer-devel. Due to firewall restrictions, outside users need to use the mail-based subscription management tools described above.
