Tcl source files are organized into 3 subdirectories of the slicer/program directory.

TCL Source Files
tcl-main
These are the files the 3D Slicer must have in order to run.
tcl-modules
These files are optional, and each one implements a specific function of the Slicer. Tcl files in this directory are referred to as modules.
tcl-shared
These files implement functionality that may be shared by multiple modules. For example, generally helpful procedures or new widgets are located here.

VTK classes are found in the slicer/vtksrc directory.

C++ Source Files
slicer/vtksrc
These are the classes that the Slicer adds to VTK.

Scripts that are used to run the Slicer and perform useful functions are found in the slicer/ directory.

Shell Scripts
slicer/
Useful shell scripts: read slicer/README.scripts for an overview.

This website is also in the slicer source code tree. You may locally generate the website and tcl code documentation yourself.

Website
slicer/doc
This directory contains files used to generate this website.
