Tcl source files are organized into 3 subdirectories of the slicer/program directory.

TCL Source Files
tcl-main
These are the files the 3D Slicer must have in order to run.
tcl-modules
These files are optional, and they contain the code to handle the modules that are accessed using the buttons on the 3D Slicer's main menu.
tcl-shared
These files may be shared by multiple modules.
