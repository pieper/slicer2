Compatibility

The 3D Slicer is compatible with Solaris 2.8 (sometimes called Solaris 5.8) and OpenGL 1.2. We do not formally support Solaris 2.5 but other users have been able to make it work with some difficulty.

Installation Steps

  1. Downloaded the desired version of the slicer.tar.gz file to your computer. FTP instructions for the most recent version were emailed to you upon receipt of your signed Software License Agreement. If you have not filled this out yet, click here. Replace slicer in the following instructions with the correct version, for example slicer-1.3.0.
     
  2. At a shell prompt, type the following to uncompress the file and create a slicer directory: 
    gunzip slicer.tar.gz
    
tar xf slicer.tar

     
  3. Edit the slicer file to set the SLICER_HOME environment variable to be the directory where the slicer directory is installed.
     
  4. Move the slicer file to a directory that is in every user's PATH environment variable. For example, this may be /user/local/bin.
     
  5. Repeat the last 2 steps with the following scripts from the slicer directory:
    • mrmlize
    • printmrml
    If you will be contributing to the development of the 3D Slicer, then also repeat the last 2 steps with these scripts:
    • comment
    • document

     
  6. Launch the slicer by typing the following at a shell prompt: 
    slicer
