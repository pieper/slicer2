How to Download

To obtain a copy of the 3D Slicer that includes all source code, please fill out the Software License Agreement.

Your submission is checked, and if all the fields are correctly filled out, you will then receive instructions via email detailing how to FTP the software.

Compatibility

The software is currently available for the following platforms:
  • Sun UltraSPARC workstations running operating system Solaris 2.8 and OpenGL 1.2
  • PCs running Windows98, WindowsNT, or Windows2000.
  • PCs running Red Hat Linux 7.3
  • Since we give out all source code, users can compile it for other platforms such as SGI.

Availability

The capabilities for automatic registration by mutual information are not included in the current public release.
