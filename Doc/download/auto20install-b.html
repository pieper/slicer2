Compatibility

The 3D Slicer is compatible with Windows95, Windows98, and WindowsNT, but it has not been tested with Windows2000. WinZip must be installed on your computer before you may begin the installation process.

WARNING

The installation may overwrite VTK or TCL/TK libraries in your Windows/System directory.

Installation Steps

  1. If you already have an older version of the slicer installed, uninstall it. To do this, from the Start menu, select Settings, then Control Panel, then Add/Remove Programs, and then 3D Slicer.
     
  2. Downloaded the slicer-pc.zip file to your computer. FTP instructions were emailed to you upon receipt of your signed Software License Agreement. If you have not done this yet, click here.
     
  3. Double-click on the file to open it into WinZip.
     
  4. Within WinZip, click the Install icon, or select Install from the Actions menu. This will run the Setup.exe program which launches the install shield.
     
  5. When the install shield completes, create a shortcut on your desktop by clicking on the Start menu followed by Programs and then drag the 3D Slicer menu entry onto your desktop.
     
  6. Click on the icon with the right mouse button to open a menu, and select the Properties option. Set the Start in: field to the directory where you store your data. This is the most convenient location to store your MRML files.
     
  7. You are now ready to run by double-clicking the shortcut you just created.

Frequently Asked Questions
What if I need more detailed instructions for installing under WindowsNT?
Look in the Developers Guide, under WinNT Install
How can I always load a particular data set?
Copy the original shortcut to create another. Then click on it once with the right (not left) mouse button to open a menu, and select the Properties option. At the end of the Target line, add the name of the MRML file you wish to open. Then click Apply.
How can I uninstall the 3D Slicer?
From the Start menu, select Settings, then Control Panel, then Add/Remove Programs, and then 3D Slicer.
