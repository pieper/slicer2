Compatibility

Slicer has been tested on a variety of Windows versions and graphics cards, but you may encounter problems depending on your system configuration. Slicer was compiled and most completely tested under Windows 2000. WinZip must be installed on your computer before you may begin the installation process.

Installation Steps

  1. If you already have an older version of Slicer installed, uninstall it. Slicer can be removed using the standard tools in the Windows Control Panel. For example, with Windows 2000, from the Start menu, select Settings, then Control Panel, then Add/Remove Programs, and then select the version of Slicer to uninstall.
     
  2. Download the PC version of Slicer onto your computer. FTP instructions were emailed to you upon receipt of your signed Software License Agreement. If you have not done this yet, click here.
     
  3. Double-click on the self extracting executable file to start the installation procedure.
     
  4. When the install shield has completed, a shortcut will appear in your Start Menu, under Programs.
     
  5. If you wish to specify a different location than the install directory in which to store your MRML files, open the shortcut and select the Properties option. Set the Start in: field to the directory where you store your data.
     
  6. You are now ready to run by clicking on Start then Programs then the Slicer shortcut.

Frequently Asked Questions
What if I need more detailed instructions for installing under WindowsNT?
Look in the Developers Guide, under WinNT Install
How can I always load a particular data set?
Copy the original shortcut to create another. Then click on it once with the right (not left) mouse button to open a menu, and select the Properties option. At the end of the Target line, add the name of the MRML file you wish to open. Then click Apply.
How can I uninstall Slicer?
Use the standard tools in the Windows Control Panel to remove the version of Slicer that you no longer wish to use.
