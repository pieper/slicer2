Compatibility

Slicer is compatible with Red Hat Linux 7.3.

System Requirements

  • Memory: Minimum: 128 MB.
  • Disk Space: 31.6 MB to download the installer, which expands to about 119 MB. The tutorial data set requires 37 MB.
  • Processor Speed: Has run successfully on a 450-MHz machine; a faster processor speed is recommended.

Installation Steps

Replace slicer in the following instructions with the correct version, for example, slicer-1.3.0.
  1. Download the desired version of the slicer.tar.gz file to your computer. (Instructions about how to download the most recent version were emailed to you upon receipt of your signed Software License Agreement. If you have not filled this out yet, click here.)
     
  2. At a shell prompt, type the following commands to uncompress the file and create a slicer directory: 
    

    gunzip slicer.tar.gz
    
tar xf slicer.tar
    


     
  3. 
Edit the slicer file to set the SLICER_HOME environment variable
to point to the directory where the slicer directory is installed.


     
  4. 
Move the slicer file to a directory that is in every user's PATH
environment variable. For example, this may be /user/local/bin.


     
  5. 
Repeat steps 3 and 4 with the following scripts from the slicer directory:

      

    • mrmlize

    • printmrml

    
If you will be contributing to the development of Slicer, then also
repeat steps 3 and 4 with these scripts:

      

    • comment

    • document

    


     
  6. 
Launch Slicer by typing the following command at a shell prompt:

    

    slicer
